# Mobius Cafe — Website

Single-page site for Mobius Cafe & Dining, Chatswood Chase (Sydney).

## Stack
Plain HTML, CSS, JS. No build step — just open `index.html` in a browser, or serve the folder.

## Files
- `index.html` — markup
- `styles.css` — styles, tokens, animations, responsive rules
- `script.js` — nav scroll state, scroll-reveal, mobile menu toggle

## Brand
Bright orange `#FF5A1F` · Black `#0B0B0B` · Cream white `#FFF8F2`.

## Local preview
```bash
python3 -m http.server 8000
# then open http://localhost:8000
```
